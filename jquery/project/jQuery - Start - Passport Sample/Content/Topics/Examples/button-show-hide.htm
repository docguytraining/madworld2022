Using Buttons to Show or Hide Divs

What this script does

When a button is clicked, the script checks the ID attribute of the button, and then hides or shows elements based on their ID attribute.

If you click a button with ID="all", then all divs will show.

If you click a button with any other ID value, then only divs with that same CLASS will be displayed. Other divs will be hidden. So to be clear, on the button element you have an ID. And on the div to show or hide, you use a CLASS that matches the button ID.

The script works on divs that are children of a div with the ID="pagetopics". That way we don't affect any other divs unintentionally.

What problem does it solve?

Helps people sort a list of divs based on role or topic or something like that.

The script

show-hide-div.js /// /// filter buttons that apply based on class /// $(function() { var $btns = $('.btn').click(function() { if (this.id == 'all') { // triggers if user clicks the button with id="all" console.log("Show All filter button clicked"); $('#pagetopics > div').fadeIn(625); } else { var $classClicked = $('.' + this.id).fadeIn(625); // triggers is user clicks any other button console.log("Filter button clicked"); $('#pagetopics > div').not($classClicked).hide(); } $btns.removeClass('active'); //removes active class from the previously-clicked element $(this).addClass('active'); // adds active class to the current element }); });

Installation Instructions

  1. Copy the scroll-top-nav.js file to your scripts folder. (Create in Content\Resoources\scripts if it doesn't exist.)

  2. At the end of your modern.css file, add the styles from the section above.

  3. Edit Content\Resources\Snippets\UI Elements\include-at-top-of-all-pages.flsnp to include a reference to this script.

    Snippets\UI Elements\include-at-top-of-all-pages.flsnp

     <body> <div id="scripts"> <script src="../../scripts/show-hide-div.js"></script>

    Remember, the file path is a link to the script relative to this Snippet file. So if the snippet file moves, or the script file moves, you will need to adjust the path here.

  4. Build your primary target and test by scrolling down this page.
