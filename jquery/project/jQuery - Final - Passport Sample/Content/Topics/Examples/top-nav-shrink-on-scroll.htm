Shrink Top-Nav Menu on Scroll

This script relies on an additional jQuery library to run. The additional library is called jQurey-UI, and is a very popular plugin.

What this script does

When you complete the steps on this page, you should be able to scroll down the page and the top-nav header will do the follwing:

  • Menu text will disappear

  • Logo will shrink

  • Search box will disappear

  • Top-nav size will shrink

What problem does it solve?

If you are using top-nav, and you have a bookmark that is down a page, when a person follows a link to that bookmark, the bookmark displays behind the top-nav bar. This can be confusing because it looks like it didn't take you where you expected. It did; you just can't tell because the header you're looking for is behind the topnav bar.

By reducing the size of the top nav, we maintain the logo and branding, but we make it much easier to see the heading that is linked to.

In addition, by removing the menu when you scroll, we leave more room for content on the page, reserving the full top-nav bar for when you are scrolled to the top of the page.

The script

scroll-top-nav.js /// CHANGE TOP MENU SIZE SMALLER WHEN SCROLLING /// REQUIRES JQUERY UI $(window).scroll(function(){ // make top menu change size when scrolling if($(document).scrollTop() > 100) { // if the window scoll goes beyond thid point, then: $('nav').addClass('small'); // add the class 'small' to the 'nav' element. } else { // if the window is NOT beyond that point, then: $('nav').removeClass('small'); // remove the 'small' class from the 'nav' element. } });

This one is invoked by a window scroll, so we don't need to call it anywhere else.

CSS needed for the script to work

modern.css

 /* CSS for scroll-top-nav.js */ .title-bar { width: 100%; position: fixed; z-index: 10; padding: 3rem; } nav.title-bar { padding: 10px 0 15px 0; height: 130px; transition: all 0.5s ease-in-out; -webkit-transition: all 0.5s ease-in-out; -moz-transition: all 0.5s ease-in-out; } nav.title-bar div.nav-search-wrapper { margin-top: 0; transition: all 0.25s ease-in-out; -webkit-transition: all 0.25s ease-in-out; -moz-transition: all 0.25s ease-in-out; } nav.title-bar div.title-bar-layout a.logo { background-size: 100%; transition: all 0.5s ease-in-out; -webkit-transition: all 0.5s ease-in-out; -moz-transition: all 0.5s ease-in-out; } nav.title-bar.small { /* This is the height of the nav with this class */ height: 25px; } nav.title-bar.small div.navigation-wrapper { /* Hide the menu */ display: none; } nav.title-bar.small div.title-bar-layout a.logo { /* Make the logo small */ background-size: 50%; margin-top: -33px; } nav.title-bar.small div.nav-search-wrapper { /* Hide the search */ margin-top: -150px; }

Installation Instructions

  1. Copy the scroll-top-nav.js file to your scripts folder. (Create in Content\Resoources\scripts if it doesn't exist.)

  2. Locate the jquery-ui folder in this project in Content\Resources\scripts\jquery-ui

  3. Copy the whole jquery-ui folder into your project in your scripts folder. The jquery-ui folder needs to be at the same level as the scroll-top.nav.js script.

  4. At the end of your modern.css file, add the styles from the section above.

  5. Edit Content\Resources\Snippets\UI Elements\include-at-top-of-all-pages.flsnp to include a reference to this script.

    Snippets\UI Elements\include-at-top-of-all-pages.flsnp

     <body> <div id="scripts"> <script src="../../scripts/scroll-top-nav.js"></script>

    Remember, the file path is a link to the script relative to this Snippet file. So if the snippet file moves, or the script file moves, you will need to adjust the path here.

  6. Build your primary target and test by scrolling down this page.
